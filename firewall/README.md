# Firewall

This project covers configuring a firewall on web servers using `ufw` (Uncomplicated Firewall).

## Task

Configured `ufw` on `web-01` to block all incoming traffic except:
- Port `22` — SSH access
- Port `80` — HTTP traffic
- Port `443` — HTTPS SSL traffic

## Usage
```bash
sudo ufw status
```
